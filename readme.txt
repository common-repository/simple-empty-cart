Are you tired of the tedious process of removing items from your cart one by one? This plugin is here to streamline your online shopping experience.

With our intuitive plugin, customers can now empty their cart with just one click, making it perfect for stores that handle large quantities of diverse products.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/simple-empty-cart` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to the WooCommerce cart page and click the "Empty Cart" button.

== Frequently Asked Questions ==

= How does this plugin work? =
It adds an "Empty Cart" button to the WooCommerce cart page. When clicked, it empties the cart.

= Is this plugin compatible with all themes? =
Yes, it should work with any WooCommerce-compatible theme.
